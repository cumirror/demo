1.安装

The MIT Scheme Interpreter can be found in lang/mit-scheme.

cd /usr/ports/lang/mit-scheme

/* 采用下面的命令一步完成 make、 make install 和 make clean 这三个分开的步骤的工作。 */

make install clean

2.运行

阅读《计算机程序的构造与解释》1.2章中的换零钱方式统计后，将如下程序

---------------------------------------------------------------------------

(define (take-kind-coin kind-coin)

(cond ((= 1 kind-coin) 1)

((= 2 kind-coin) 5)

((= 3 kind-coin) 10)

((= 4 kind-coin) 25)

((= 5 kind-coin) 50)))

(define (change count kind-coin)

(cond ((= 0 count) 1)

((or (= 0 kind-coin) (< count 0)) 0)

(else (+ (change (- count

(take-kind-coin kind-coin))

kind-coin)

(change count

(- kind-coin 1))))))

(define (change-method count) (change count 5))

---------------------------------------------------------------------------

保存到change.scm文件中。

cd到文件所在目录，在shell中执行：

scheme -compiler

进入scheme后，可以在解释器中直接敲入lisp程序，也可以编译运行保存在文件中的lisp源程序。

(cf "change") //将编译change.scm，产生change.com文件

(load "change") //将load change.com文件

(change-method 100)

注：1.只有带上“-compiler”选项，才能使用cf编译.scm（lisp文件扩展名）;

2.运行一个过程时，要带上括号；

3.上面的程序，如果我将(define (change-method count) (change count 5)) 放到第一行，无法执行change-method，只能执行最后的过程change count kind-coin，为啥啊？
